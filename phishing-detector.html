Log Analyzer

About the Project

This is a detailed description of the Log Analyzer project. It was built to solve a specific problem by leveraging modern technology. The application reads various log file formats, parses them for critical information like errors and warnings, and then utilizes a Generative AI model to interpret these errors.

The AI not only provides a human-readable explanation of the error but also suggests potential solutions and code fixes. The long-term vision for this project is to create a fully automated system that can apply the suggested fixes, run tests, and ensure system stability with minimal human intervention.

Technologies Used

  • Python
  • LangChain
  • OpenAI API
  • Streamlit
